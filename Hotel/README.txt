1. Új mappa létrehozása a htdocs mappában.
2. XAMPP megnyitása, Apache, MYSQL elindítása a start gombbal.
3. A localhost/phpmyadmin felületen egy új tábla létrehozása lehetőleg hotel_project -néven de ez módosítható a későbbiekben.
4. A tábla létrehozása után importálás fülre kattintás. Ott az importálandó fájl menüpontban a fálj kiválasztására kattintunk.
5. A db mappában található hotel_project.sql fálj tallózása.
6. Egy böngészőben a keresőbe az alábbi url írása: http://localhost/x/ x helyére a htdocsban lévő mappanév kerüljön.
